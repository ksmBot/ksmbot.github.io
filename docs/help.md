### AB游戏规则介绍

#### 机制

随机生成一个自然数(以免你不记得，*首数字不能为0*(在游戏的全程中,这一个前提一直成立))，玩家猜数，每个位置正确的数字记为1个**A**，位置不正确但存在的数字记为1个**B**，目标为猜出原数字

示例:

```
1234 猜 1324 为 2A2B
```

因为1和4在原位置而2和3不在原位但存在于原数

```
1234 猜 3456 为 0A2B
```

同理,你可以自行举例并计算

#### 需要注意

你可以猜的次数为2*长度+2(一般是这样的)，推荐的数字长度为3~6

### 指令

下面的指令均为大小写敏感，也就是说大小写错了就不能触发了

#### 开始

```
/ab+特殊模式码[见下文特殊模式说明]+空格+数字[代表位数,缺省为4]
```

支持的位数值为2~10

---

示例:

```
/ab
```

开始一次**4长度**的游戏

```
/AB 2
```

开始一次**2长度**的游戏

注意不能一个字母大写一个字母小写

```
/abbalabala 8balabala
```

没错,这样也能识别,开始一次**8长度**的游戏

实际上程序识别到的有效内容为/ab~~balabala~~ 8~~balabala~~也就是`/ab 8`


#### 结束

- 最简单的方法(但是可能会和其他bot/功能冲突)

直接输入`/end`

- 不需要那么麻烦的方法(但是可能会和其他bot/功能冲突)

任何`end`或`End`开头的输入

- 最安全的方法

任何包含`abend`的输入

---

示例:

```
/end
ender               //有效信息只有"end"
End Time            //有效信息只有"End"
114514abend1919810  //有效信息只有"abend"
```

#### 召唤这个界面

```
/abhelp
```

#### 个人游戏统计

```
/abme
```

包括你的总猜次数，正确次数等个人信息

一般来说，普通模式Acc达到45%就算优秀，如果你喜欢挑战自我，那么35%也很强了

#### 特殊模式

如果要进入特殊模式，就在`/ab`后面加上代表模式的字符

不支持多个字符连用，会被识别为普通模式

`n`或者`N`或者*空*或者*其他识别不了的东西* - 普通模式

`u`或者`U` - [支援] **Unsame**模式，目标数每位数字值各不相同，可猜次数为**2*长度**

`t`或者`T` - [支援] **Tips**模式，每次猜数65%显示一个Tips(可能会给你一些有用的提示)，可猜次数为**长度+1**

`e`或者`E` - [支援] **Easy**模式，显示每个AB的位置，可猜次数为**长度**

`a`或者`A` - [挑战] **Addtion**模式，每次猜数后目标数将+1，可猜次数为**2*长度+4**

`c`或者`C` - [挑战] **Chaos**模式，10%AB颠倒，10%输出乱码，可猜次数为**2*长度+5**

`r`或者`R` - [挑战] **Random**模式，输入有25%概率被随机，可猜次数为**2*长度+(0~6)**(它是随机的)

`h`或者`H` - [创世] **What The Hell**模式，上面三个挑战加起来，可猜次数为**3*长度+4**

---

示例:

```
/aba 6
```

开始一次**6长度**且为**Addtion**模式的游戏
