## 邦多利猜猜看

使用 <code>cck</code>就可以开始对邦邦卡面的cck，由kumo的koishi插件提供技术支持。

注意：数据初始化使用来自bestdori的数据支持，如果由于服务器网络波动导致的无法正常获取资源，可能导致不断地卡在初始化，不停地猜同一张卡面等问题。

此时请尝试使用<code>cck -R</code>重置缓存后等待服务器重新下载，若多次失败可能服务器暂时不正常，请稍后重试。

## ksm图库

输入

&emsp;

<code>ksm图库</code>
 
&emsp;

此方法图片来自Pixiv，但画质较低，需自行下载原图

或

<code>二次元图库</code>

此方法图片来自韩小韩api

即可获取图片





## 音乐

这个搜不到vip曲子哦

<code>

#音乐 关键词
自动搜索所有源以找出来找最佳音频来源

#外链 关键词
自动搜索所有源，以外链信息的形式发出

#QQ 关键词
搜索QQ音乐

#网易 关键词
搜索网易云音乐

#网易电台 关键词
搜索网易云电台，一般来说是直接选择找到的第一个节目，但是关键词可以以 “电台名称|节目名称”的格式指定电台节目

#酷狗 关键词
搜索酷狗音乐
</code>
